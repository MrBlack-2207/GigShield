# GigShield Frontend Demo

Minimal React demo app for Phase 2 using only `/api/contract` endpoints.

## Run

```bash
cd frontend
npm install
npm run dev
```

Dev server runs on `http://localhost:3000` and proxies `/api/*` to `http://localhost:8000`.

## Demo Flow

1. Register worker on **Onboarding**
2. Quote and purchase on **Policy**
3. Use hidden/demo controls on **Dashboard**
4. Review payouts on **Claims**
5. Cashout on **Wallet**
